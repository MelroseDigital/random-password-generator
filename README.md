# random-password-generator
A simple and secure Random Password Generator built with Python. This project generates strong, customizable passwords using a mix of uppercase letters, lowercase letters, numbers, and special characters to enhance security.

# Features
Generate secure random passwords

Customizable password length

Option to include:

Uppercase letters (A–Z)

Lowercase letters (a–z)

Numbers (0–9)

Special characters (!@#$%^&*, etc.)

Lightweight and easy to use

Beginner-friendly Python project
